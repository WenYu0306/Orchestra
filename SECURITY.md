# 安全策略

## 报告漏洞

如果你发现安全漏洞，请**不要**在公开 Issues 中报告。

请通过邮件联系：`WenYuli0306521@outlook.com`

请包含：

- 漏洞的详细描述
- 复现步骤
- 受影响的版本
- 可能的修复建议（如有）

我会在 48 小时内回复确认收到，并在 7 天内提供修复方案。

## 支持版本

| 版本 | 支持 |
|------|------|
| 1.0.x | ✅ |

## 依赖安全

本项目依赖 `nodriver`、`FastAPI`、`httpx` 等第三方库。

- `.env` 文件不要提交到 Git（已在 `.gitignore` 中排除）
- `config.yaml` 中的个人信息字段建议填写假数据或留空
- 不要在公开场合分享你的 `data/applied_jobs.json`，其中包含你的简历匹配结果

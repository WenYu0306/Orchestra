## 核心数据

| 指标 | 数值 |
|------|------|
| 搜索词 | 8 个（AI应用开发、LLM、全栈开发...）|
| 每词职位 | 稳定 15 个 |
| 评分分化 | 82 / 72 / 68 / 55 / 45 |
| 分层 | high / medium / try / skip / pending |
| 架构验证 | 决策层在 7 次执行层切换中未修改 |
| 代码量 | 后端 11 文件 + 前端 5 组件 = 2300+ 行 |

## 关键决策记录

1. **Playwright → 反爬检测 webdriver 标识，页面跳 about:blank**
2. **DrissionPage → 选择器语法不稳定，控制粒度不足**
3. **CDP 直连 → 端口管理复杂，进程生命周期问题**
4. **nodriver → 连接稳定但无法触发 BOSS React 事件系统**
5. **截图 + Qwen VL → JD 提取不稳定，300-400 字偏差大**
6. **DOM 遍历 → 文本全是页面 footer，无法定位右侧面板**
7. **httpx 调 BOSS API → 从 Chrome cookie 冒充登录用户, 15 职位/搜索**

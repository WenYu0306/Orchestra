# JobHunter — 三层 Agent 架构

```
┌─────────────────────────────────────────────────────────────┐
│          决策层 · DeepSeek Orchestrator                      │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────┐ ┌──────┐   │
│  │ JD 评分   ││ 分层决策  │ │ 招呼语   │ │假检测│ │待选区│  │
│  │ (0-100)  │ │高≥80中≥60│ │ 动态生成  │ │      │ │最多5 │  │
│  └──────────┘ └──────────┘ └──────────┘ └──────┘ └──────┘  │
├─────────────────────────────────────────────────────────────┤
│          执行层 · 数据获取                                    │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────────┐   │
│  │ API 直取  │ │ 详情富化  │ │ 标签兜底  │ │ 翻页/保活     │   │
│  └──────────┘ └──────────┘ └──────────┘ └──────────────┘   │
├─────────────────────────────────────────────────────────────┤
│          连接层 · Chrome Bridge                               │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────────┐   │
│  │nodriver  │ │ CDP 协议  │ │ Cookie   │ │ 反爬对抗      │   │
│  │启动Chrome│ │evaluate   │ │ 管理     │ │webdriver     │   │
│  └──────────┘ └──────────┘ └──────────┘ └──────────────┘   │
└─────────────────────────────────────────────────────────────┘

数据流：Chrome → API JSON → 标签评分 → top 30 详情重评 → 分层 → SSE 推送
```

## 核心数据

| 指标 | 数值 |
|------|------|
| 搜索词 | 22 个（动态生成 + 配置预设） |
| 搜索城市 | 北京、长春 |
| 岗位数据 | 660+ 岗位/次 |
| 评分方案 | 标签快速初筛 + 详情 API 重评 top 30 |
| 最终输出 | 20 个岗位按 5/7/8 三层分层 |

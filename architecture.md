## 核心数据

| 指标 | 数值 |
|------|------|
| 搜索词 | 18 个（动态生成 + 配置预设） |
| 搜索城市 | 北京、长春（动态获取 BOSS 城市 API 编码） |
| 岗位数据 | 500+ 岗位/次 |
| 评分方案 | 标签快速初筛 + 详情 API 重评 top 30 |
| 评分分化 | 35-92 分（详情重评后） |
| 最终输出 | 20 个岗位按 5/7/8 三层分层，JD 优先 |
| 真发送 | APPLY_LIMIT 环境变量控制，10 秒倒计时可取消 |
| 测试覆盖 | validator 10 个单元测试全绿 |

## 设计理念

设计方向由 AI 设计工具 Ardot 生成，基于项目功能需求而非预设模板。核心理念：

- **功能主义优先。** 设计服务于"一眼就知道 Agent 在干嘛"。进度条和统计卡片数字跳变比花哨动效更重要。信息层级靠字重和间距区分，不靠颜色堆砌
- **干扰最小。** 状态灯平时安静呼吸，遇到验证码才变红脉冲。底部通知栏条件显示——验证码红色（不自动消失）、完成摘要蓝色（3 秒消失）、错误橙色。只在需要人工介入时才用颜色抢夺注意力
- **配色系统。** 浅色基底（#F7F5FA / #F7F8FA），Geist 字体族，三层岗位卡片用靛蓝/蓝/琥珀三组颜色区分。靛蓝 = 高匹配（≥80）、蓝 = 中匹配（60-79）、琥珀 = 可尝试（40-59）
- **毛玻璃面板。** `backdrop-filter: blur(24px)` 半透明卡片 + 多层阴影，卡片 hover 微抬 2% + 左侧靛蓝色条淡入。骨架加载卡 shimmer 呼吸，入场动效淡入上浮
- **Aurora 光晕背景。** 三个固定模糊光球在四角缓慢漂移，纯装饰。12% 透明度，不干扰可读性
- **框架中立。** Vue 3 单页，无路由，组件接口独立于后端。SSE 数据流与 UI 解耦——换掉 mock 逻辑不需要重写组件。以后加多平台多用户只改数据层
